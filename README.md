# BarbShop

 Landing page moderna para barbearia desenvolvida em React + TypeScript + Vite.

## 🚀 Tecnologias

- React 18
- TypeScript
- Vite
- CSS3

## 📦 Instalação

```bash
npm install
npm run dev
```

## 🔨 Build

```bash
npm run build
```

## 🌐 Deploy

Deploy automático via GitHub Pages.

## 📱 Features

- Design moderno e responsivo
- Animações suaves
- Seções completas: Hero, Serviços, Sobre, Galeria, Contato
- Formulário de contato
- Menu mobile responsivo
